# Phase 2 — Optional TrOCR dependencies for the OCR service.

# Install alongside requirements.txt when you want handwritten recognition:
#     pip install -r requirements.txt
#     pip install -r requirements-ocr.txt

#   - CI runners on low-RAM machines can stay slim
